Agent Quest

Learn About Agentic AI Through Play

You are an AI agent navigating a world of challenges. Deploy specialized subagents to solve obstacles that require specific capabilities!

How to Play:

  • Move: Arrow keys or WASD
  • Deploy Subagents: Number keys 1-4

Subagent Types:

1 - Vision Agent (Blue): Identifies and analyzes visual patterns
2 - Logic Agent (Green): Solves puzzles and logical challenges
3 - Navigation Agent (Yellow): Finds optimal paths through mazes
4 - Security Agent (Red): Bypasses security barriers
